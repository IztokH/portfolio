NEO FFI – Fullstack SaaS Platform

A fullstack web application developed using HTML, CSS, JavaScript, Node.js, Express, and MySQL. The platform enables user and company management, data processing, and dynamic result visualization, including chart-based analytics and PDF export. Built with a structured backend architecture, role-based access control, and integration of external services for automation and scalability.
